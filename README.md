# Proyecto-clinica-veterinaria-patitas-felices-
#Descripcion 
Proyecto universitario desarrollado en grupo para la gestion basica de una clinica veterinaria llamada "Patitas Felices"
#Objetivo del proyecto
Aplicar conocimientos adquiridos en el curso de introduccion a la programacion Java POO mediante el desarrollo colaborativo de un sistema funcional
#Tecnologías utilizadas
- Lenguaje: Java 
- Control de versiones: Git y GitHub
- Entorno de desarrollo: NetBeans
